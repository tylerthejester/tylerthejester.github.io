Hello everyone! I can show you how to create a personal web page.

How to create a personal web page:

1. Make an account on GitHub

2. Create a new repository called [put-your-username-here].github.io

3. Download a code editor (Visual Studio Code is one option for this)

4. Create an index.html file in the code editor (you may also want to use a CSS file for styles, and a JS file for functions)

5. Add images if needed

6. Upload to GitHub repository

7. Commit

8. Test your website URL
